# 📦 Install Node.js First (Required)

## ⚠️ Node.js is Not Installed

You need to install Node.js before running the app. Node.js includes `npm` (Node Package Manager) which is required.

---

## 🚀 Quick Installation Guide

### Step 1: Download Node.js

1. **Go to Node.js website:**
   - Visit: https://nodejs.org/
   - Or directly: https://nodejs.org/en/download/

2. **Download the LTS Version (Recommended)**
   - Click the big green button that says "LTS" (Long Term Support)
   - This is the stable version
   - It will download a `.msi` file (Windows installer)

### Step 2: Install Node.js

1. **Run the Installer**
   - Double-click the downloaded `.msi` file
   - Click "Next" through the installation wizard
   - ✅ **IMPORTANT**: Check "Automatically install necessary tools" if asked
   - Click "Install"
   - Wait for installation to complete

2. **Verify Installation**
   - Close and reopen your terminal/PowerShell
   - Run these commands to verify:
     ```bash
     node --version
     npm --version
     ```
   - You should see version numbers (like `v20.10.0` and `10.2.3`)

### Step 3: Restart Your Computer (Recommended)
- This ensures Node.js is properly added to your system PATH
- After restart, you can proceed to run the app

## 🆘 Troubleshooting

### "node is not recognized" after installation
- **Restart your computer** (this usually fixes it)
- Or restart your terminal/PowerShell
- Check if Node.js is in your PATH environment variable

### Installation Fails
- Make sure you have administrator rights
- Try running the installer as Administrator
- Check Windows Defender isn't blocking it

### Still Having Issues?
- Visit: https://nodejs.org/en/download/
- Download the Windows Installer (.msi)
- Make sure to select "Add to PATH" during installation

- **npm**: Package manager (installs project dependencies)

Both are free and open-source!
