# Quick Start Guide

## Installation Steps

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Start Development Server**
   ```bash
   npm run dev
   ```

3. **Open Browser**
   Navigate to `http://localhost:5173`

## Using the App

### Home Page
- Enter your destination in the search bar
- Click "Search" to start planning

### Trip Planner Page
1. **Enter Locations**: Add your current location and destination
2. **Select Trip Duration**: Choose how many days you're planning
3. **Choose Transport**: Select own vehicle or public transport
4. **View Information**:
   - Route map and directions
   - Weather forecast
   - Traffic conditions
   - Nearby places to visit
   - Hotel recommendations
   - Destination photos and attractions

### To-Do List Page
- Add items you need for your trip
- Check off items as you pack
- Use suggested items for quick additions
- Track your progress
